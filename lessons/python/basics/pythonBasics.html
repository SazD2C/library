Basics of Python

Variables

Variables are a way to store data within python. It tells your computer that hey this word has data inside of it.

x = "hello"

This is how python states a variable. This code lines tell the system that "X" has the word "hello" in it. Now we can say x, and the system will tell us "hello". This is how variables work, they store information, so that it may be used later.

There are 3 different types of variables

Strings

Strings are words, and letters. When we declare a string within python we have to wrap it up in "", or the system won't understand and toss an error. For example the code above is a string in a variable, as notated by the ""

Ints

In the coding world these are refered to as ints, but their real name is integers. (It means numbers) This is shown in the code below.

x = 10

Here we have set x to equal the number 10. We don't have to wrap it in "" because it is not a string.

Boolean

A boolean means true or false. That's it. Easy right?

x = True

Now since we are in python it is mandatory that you capitalize the T in true and F in false, or the system won't know it is a boolean.

But how do we access variables? The easiest way todo this is by using the "print()" function. What this does is it prints what ever is inbetween () to the console!

print(x)

We can print regular strings by putting words in a "" within the () of print. For example:

print("Hows it flowin")

You can also print ints without wrapping them in "", but it doesn't matter

Conditionals

None of this matters though without Condentionals. A condentional sets a condition for the program to meet, for example what do we do if we want to check if x equals the word "hey", and if so print "hello"?

Well we would do this: 


x = "hey"
if(x=="hey"):
  print("hello")

First we set x to equal hey, then we use an if statement to check if x equals hey or not. Take careful note of how we do this.

We state if(variable==whatWeWannaCheck): if() tells the program that we're doing a if statement here. variable is the variable we are checking, and whatWeWannaCheck is the condition we are looking to meet. It can be a boolean, string, int, or even another variable. Then we add : at the end to tell the program that the next INDENTED lines are to be ran if the condition is met.

Now indented is in caps because indentation is incredibly important in python. Lets take a look at a piece of code to prove this. 


x = 1
if(x==1):
  print("hey that is right!")
if(x==2):
  print("Hey that is wrong...")

The indentions here tell the program that the prints belong to a specific if statement... so what happens if we do not indent?

x = 1 if(x==1): print("hey that is right!") if(x==2): print("Hey that is wrong!")

Now this code just prints hey thats right and wrong! What happens if this is the code for a password checker? Does this imply that your password is correct and incorrect? See how important indents are?
